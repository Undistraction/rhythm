    rhythm

    @mixin rhythm($font-size, $lines, $offset, $from-context) { ... }
    View source

    Description

    Adjust a block to have a different font size and line height to maintain the rhythm. $lines specifies how many multiples of the baseline rhythm each line of this font should use up. It does not have to be an integer, but it defaults to the smallest integer that is large enough to fit the font. Use $from-context to adjust from a font-size other than the base font-size.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $font-size

    The font-size for this block

    		String—none
    $lines

    The number of rhythm units (lines) each line of text should should fill.

    		Number—none
    $offset

    (0) Value to be removed from the total line-height (allowing you to adjust for borders).

    		Number—none
    $from-context

    The context of the block (when using ems)

    		Number—none

    Output

    Output the calculated font-size and line-height.

    Requires

    • [mixin] rhythm-render-property
    • [mixin] rhythm-render-leading
    • [function] validate-font-size
    • [function] rhythm-convert-value
    • [function] rhythm-lines-for-font-size
    • [variable] rhythm-render-unit

    Author

    • Pedr Browne (extracted from Compass Vertical Rhythm by Eric M. Suzanne).

    rhythm-context

    @mixin rhythm-context($new-base-font-size, $new-base-line-height) { ... }
    View source

    Description

    Use a different rhythm for a block by temporarily changing the configuration values '$rhythm-base-font-size' and '$rhythm-base-line-height'. While this will do nothing by itself, it can be combined with rhythm-establish to change the global rhythm temporarily, for example at a breakpoint.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $new-base-font-size

    The new base-font-size to use for the content of this mixin

    		Number—none
    $new-base-line-height

    The new base-line-height to use for the content of this mixin

    		Number—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Requires

    • [variable] rhythm-base-font-size
    • [variable] rhythm-base-line-height

    Author

    • Pedr Browne (extracted from Compass Vertical Rhythm by Eric M. Suzanne).

    @function rhythm-convert-value($length, $to-unit, $from-context, $to-context) { ... }
    View source

    Description

    Convert any CSS length or percentage value to any another.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $length

    A css length or percentage value

    		String—none
    $to-unit

    String matching a css unit keyword, e.g. 'em', '%', etc.

    		String—none
    $from-context

    ($rhythm-base-font-size) When converting from relative units, the absolute length (in px) to which $length refers (e.g. for $lengths in em units, would normally be the font-size of the current element).

    		String—none
    $to-context

    ($from-context) For converting to relative units, the absolute length in px to which the output value will refer. Defaults to the same as $from-context, since it is rarely needed.

    		String—none
